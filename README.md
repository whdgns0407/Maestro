# 마에스트로 거래소
우후 죽순 생겨나는 가상화폐 신생 거래소의 위험성을 알리기위해 만들어진 웹사이트입니다
해당 웹 사이트의 개발로 신생 거래소의 위험성을 알리는 커뮤니티(코인판, 코박) 게시글의 Total조회수가 20만회 이상 달성하였습니다.

* 코인판 주소 (로그인 후 확인가능) : https://coinpan.com/coin_info/118041119
